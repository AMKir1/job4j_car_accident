# job4j_car_accident
Приложение "Автонарушители".

1) Две роли. Обычные пользователи и автоинспекторы.

2) Пользователь добавляет описание автонарушения. 

3) В заявлении указывает: 
    * Адрес
    * Номер машины
    * Описание нарушения
    * Фотографию нарушения

4) У заявки есть статус:
    * Принята
    * Отклонена
    * Завершена
